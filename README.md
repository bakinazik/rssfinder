## Features:

- Detects rss and atom feeds automatically when you visit a page.
- Works on single-page apps and dynamic sites.
- Shows feed count on the extension badge.
- One-click copy of feed urls to clipboard.
- Caches results per tab for instant access.
- Supports YouTube, Reddit and standard blog feeds.
